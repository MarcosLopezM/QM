\begin{problema}
	Muestra la siguiente igualdad usando las propiedades de las matrices de Pauli.
	Considera el vector de matrices de Pauli \(\bm{\sigma} = (\sigma_{x}, \sigma_{y},
	\sigma_{z})\)
	\begin{equation*}
		\mathrm{e}^{i \dotprod{\op{\sigma}}{\op{n}}} = \cos\theta + i \dotprod{\op{\sigma}}{\op{n}}
		\sin\theta,
	\end{equation*}

	donde \(\op{n}\) es un vector unitario en dirección arbitraria. \emph{Hint: Usa índices
		para expresar el producto punto}.
\end{problema}
